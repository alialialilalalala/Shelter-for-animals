# 🐾 AnimalShelterAI

**Автоматизированная информационная система для учёта деятельности приюта для животных**

---

## 📋 О проекте

AnimalShelterAI — это десктопное приложение для комплексной автоматизации работы приюта для животных. Система позволяет вести учёт животных, обрабатывать заявки на усыновление, управлять задачами волонтёров, вести медицинские карты, учитывать пожертвования и формировать отчёты.

### Основные возможности

- 🔐 **Авторизация и регистрация** с хешированием паролей и опцией "Запомнить меня"
- 👤 **Ролевая модель доступа** (5 ролей: Администратор, Менеджер, Ветеринар, Волонтёр, Пользователь)
- 🐕 **Управление животными** (CRUD, фильтрация, поиск, загрузка фото)
- 📋 **Заявки на усыновление** (создание, просмотр, одобрение/отклонение)
- ✅ **Задачи волонтёров** (создание, назначение, выполнение, отклонение)
- 🏥 **Медицинские записи и вакцинации** (полная история лечения)
- 👥 **Управление пользователями** (добавление, редактирование, блокировка)
- 💰 **Учёт пожертвований** (денежных и материальных)
- 📊 **Формирование отчётов в Excel** (по животным, заявкам, финансам, волонтёрам)
- 👤 **Личный кабинет пользователя** (профиль, история заявок)
